# Decline-in-reading-among-young-people

# Analyse de la corrélation : Temps d'écran vs Lecture (18-24 ans)

Ce projet analyse l'impact des habitudes numériques sur le temps de lecture chez les jeunes adultes.

##  Visualisations incluses
- **Répartition par activité** (Réseaux sociaux, Études, Divertissement).
- **Évolution par âge précis** (Focus sur la tranche 18-24 ans).
- **Corrélation** entre la consommation numérique et le temps passé à lire.

##  Technologies utilisées
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

##  Contenu du dépôt
- `daily_internet_usage_by_age_group.csv` : Données sur l'usage d'internet.
- `reading_habits_students.csv` : Données sur les habitudes de lecture.
- `analyse.ipynb` : Le notebook contenant tout le code et les graphiques.
